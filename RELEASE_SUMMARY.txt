📦 NEW FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. CONFIGURATION PERSISTENCE SYSTEM
   • NVS-based storage for AP and Station configurations
   • Survives reboots and power cycles
   • Base64 password encryption
   • Auto-start/auto-connect on boot
   • Serial commands for configuration management

2. WEB CONFIGURATION INTERFACE
   • Professional /config page for settings management
   • AP configuration form (SSID, password, channel, auto-start)
   • Station configuration form (SSID, password, auto-connect)
   • Password masking (never displayed in plain text)
   • Real-time validation and error handling
   • Clear all configurations option

3. REBOOT MODAL SYSTEM
   • 10-second countdown timer
   • Confirm/Cancel buttons
   • Visual progress display
   • Automatic page reload after reboot
   • User control over reboot operation

4. INSTANT MODE SWITCHING
   • Toggle between AP and Station without reboot
   • 2-second operation vs 10+ second reboot
   • Uses saved configurations automatically
   • Web UI buttons for easy switching
   • Error handling for missing configurations

5. RESPONSIVE NAVIGATION MENU
   • Desktop: Horizontal navigation bar
   • Mobile: Hamburger menu (☰)
   • Touch-friendly (44px minimum targets)
   • Smooth animations
   • Adapts to all screen sizes

6. BASE64 PASSWORD SECURITY
   • All passwords encoded before NVS storage
   • Uses ESP32's built-in mbedtls library
   • Transparent encoding/decoding
   • Secure password retrieval

🔧 NEW SERIAL COMMANDS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AP CONFIGURATION:
  ap config <ssid> <password> [channel] [auto]  - Save AP configuration
  ap config load                                 - Display saved config
  ap config clear                                - Clear saved config
  ap start                                       - Start with saved config

STATION CONFIGURATION:
  station config <ssid> <password> [auto]       - Save Station config
  station config load                            - Display saved config
  station config clear                           - Clear saved config
  station connect                                - Connect with saved config

🚀 QUICK START GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

BUILD & FLASH:
  $ pio run -e esp32dev -t upload
  $ pio run -e adafruit_feather_esp32s3_tft -t upload

SERIAL CONFIGURATION:
  > ap config "MyHotspot" "SecurePass123" 6 auto
  > station config "HomeWiFi" "MyPassword" auto

WEB CONFIGURATION:
  1. Start AP mode: mode ap
  2. Connect to ESP32 AP
  3. Open browser: http://192.168.4.1/config
  4. Configure and save settings
  5. Optional: Reboot via modal

INSTANT MODE SWITCH:
  1. Visit: http://<device-ip>/config
  2. Click "Switch to AP" or "Switch to Station"
  3. Mode changes in ~2 seconds
