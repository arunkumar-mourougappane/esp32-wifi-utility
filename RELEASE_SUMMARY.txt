                              🚀 NEW FEATURES
================================================================================

1. SIGNAL STRENGTH MONITOR
   ──────────────────────────────────────────────────────────────
   ✓ Live current connection signal strength (3s auto-refresh)
   ✓ Nearby networks scanner with signal quality indicators
   ✓ Auto-scan mode: 5, 10, 15, 30, 60 second intervals
   ✓ Hidden network detection: <Hidden Network>
   ✓ Visual quality indicators: 🟢🟡🟠🔴
   ✓ RSSI to quality percentage conversion (0-100%)
   ✓ Quality text: Excellent/Good/Fair/Weak/Very Weak
   ✓ Non-blocking scans - page content remains visible
   ✓ Real-time countdown timer for next scan
   
   Web Interface:  http://<device-ip>/signal
   Documentation:  docs/SIGNAL_STRENGTH_MONITOR.md
   Integration:    Analysis Dashboard (cyan card)

2. PORT SCANNER  
   ──────────────────────────────────────────────────────────────
   ✓ Common Ports scan (16 ports, ~20-30 seconds)
   ✓ Well-Known Ports scan (1-1024, ~10-30 minutes)
   ✓ Custom Range scan (user-defined ports)
   ✓ All Ports scan (1-65535, ~8-12 hours)
   ✓ Service identification: 25+ common services
   ✓ Real-time progress bar with percentage
   ✓ Background scanning (non-blocking TCP tests)
   ✓ Results display: Port/Service/Status table
   ✓ Gateway IP auto-population
   ✓ Security warnings and legal disclaimers
   
   Web Interface:  http://<device-ip>/portscan
   Documentation:  docs/PORT_SCANNER.md
   Integration:    Analysis Dashboard (purple card)
   
   Services Detected:
   • Web: HTTP, HTTPS, HTTP-Alt
   • Remote: SSH, Telnet, RDP, VNC
   • Email: SMTP, POP3, IMAP
   • Databases: MySQL, PostgreSQL, MongoDB, Redis
   • File Transfer: FTP, SMB
   • And 15+ more services

3. CHANNEL GRAPH VISUALIZATION
   ──────────────────────────────────────────────────────────────
   ✓ HTML5 Canvas-based bar chart (1000x400px responsive)
   ✓ Color-coded congestion levels:
     • Green (0-40%): Low congestion - Excellent choice
     • Yellow (40-70%): Medium congestion - Acceptable  
     • Red (70-100%): High congestion - Avoid
   ✓ Network count overlays on each channel bar
   ✓ Recommended channels marked with stars (⭐)
   ✓ Best channel automatic identification
   ✓ Professional graph: grid, axes, labels
   ✓ Interactive legend with explanations
   ✓ Non-overlapping channel indicators (1, 6, 11)
   ✓ Responsive design with window resize support
   ✓ Educational content about channel overlap
   
   Web Interface:  http://<device-ip>/channel/graph
   Documentation:  docs/CHANNEL_GRAPH.md
   Integration:    Button on Channel Analysis page

4. TFT DISPLAY SUPPORT
   ──────────────────────────────────────────────────────────────
   Hardware Support:
   • Adafruit Feather ESP32-S3 TFT
   • Adafruit Feather ESP32-S3 Reverse TFT
   • 135x240 pixel display, ST7789 driver
   
   AP Mode Display:
   ✓ Green "AP Mode" text indicator
   ✓ QR code for instant WiFi connection
   ✓ SSID and password display
   ✓ AP IP address
   ✓ Optimized layout for small screen
   
   Station Mode Display:
   ✓ Blue "Station Mode" text indicator
   ✓ Connection status with visual feedback
   ✓ Connected network SSID
   ✓ Local IP address
   ✓ Real-time signal strength (RSSI in dBm)
   ✓ Color-coded signal quality:
     • Green: Strong signal (-60 dBm or better)
     • Yellow: Medium signal (-60 to -70 dBm)
     • Red: Weak signal (below -70 dBm)
   ✓ Visual signal quality bar graph (0-100%)
   
   Features:
   • Automatic mode detection using esp_wifi_get_mode()
   • Dynamic display updates based on WiFi mode
   • QR code generation for AP connection
   • Real-time signal visualization
   • Professional color scheme and layout

================================================================================
                            ⚠️  IMPORTANT NOTES
================================================================================

PORT SCANNER DISCLAIMER:
  ⚠️  LEGAL AND ETHICAL USE REQUIRED:
  • Only scan devices you own or have explicit permission to scan
  • Unauthorized port scanning may be illegal in your jurisdiction
  • Comply with organizational security policies
  • Check local laws before scanning
  • Use responsibly and ethically

  NETWORK IMPACT:
  • Port scanning generates network traffic
  • May trigger IDS/IPS alerts
  • Can affect network performance
  • Should be scheduled during maintenance windows

TFT DISPLAY REQUIREMENTS:
  • Only works on Adafruit Feather ESP32-S3 TFT boards
  • Regular ESP32 boards do not have built-in display
  • Reverse TFT variant also supported
  • Libraries automatically included in TFT builds

CHANNEL GRAPH BROWSER REQUIREMENTS:
  • Requires HTML5 Canvas support
  • Modern browsers: Chrome, Firefox, Safari, Edge
  • Mobile browsers supported
  • Internet Explorer NOT supported
